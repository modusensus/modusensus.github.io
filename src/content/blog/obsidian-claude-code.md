## 为什么需要一套知识管理系统

跨考公共管理对我来说意味着从零开始学一门全新的学科。不同于本专业，公管的知识体系庞杂，涉及行政学、政策分析、公共经济学等多个板块，如果只是把笔记堆在一起，复习时会非常低效。

所以我决定在备考初期就把知识库结构搭好。

## 文件夹结构

```
MyBrain/
├── 00-Inbox/        # 未整理的临时笔记
├── 01-考研/
│   ├── 公共管理学/
│   ├── 行政法/
│   └── 每日考研日报/
├── 02-Projects/     # 毕设、博客等
└── 03-Resources/    # 参考书目录
```

## Claude Code 的使用方式

我主要用 Claude Code 做两件事：

1. **整理笔记结构** — 把零散的阅读笔记按知识点重新分类
2. **生成复习提纲** — 根据已有笔记生成章节摘要和填空题

效果出乎意料地好，特别是在梳理"新公共管理 vs 新公共服务"这类容易混淆的对比概念时。
